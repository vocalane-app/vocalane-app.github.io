# 利用規約

最終更新日: 2026年4月30日

## 1. サブスクリプション

- VocaLane Plus は自動更新サブスクリプションです
- 月額プランまたは年額プランからお選びいただけます
- 7日間の無料トライアルが含まれます
- 無料トライアル終了後、選択したプランの料金が Apple ID に請求されます
- サブスクリプションは各期間の終了日の24時間前までに解約しない限り、自動的に更新されます

## 2. 解約方法

サブスクリプションはいつでも解約できます。

iPhoneの「設定」>「Apple ID」>「サブスクリプション」>「VocaLane Plus」>「サブスクリプションをキャンセル」

解約後も、現在の課金期間の終了日まで Plus 機能をご利用いただけます。

## 3. 免責事項

本アプリは音楽練習の補助を目的としています。練習成果を保証するものではありません。

## 4. 準拠法

本規約は日本法に準拠します。
